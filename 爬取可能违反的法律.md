### 一、民事责任及行业自治规范
| 情形        | 行为描述                       | 法律依据补充                                                                                                        |
| :-------- | :------------------------- | :------------------------------------------------------------------------------------------------------------ |
| **不正当竞争** | 违反Robots协议、实质性替代产品、妨碍正常经营。 | **《中华人民共和国反不正当竞争法》第十二条**：经营者不得利用技术手段，通过影响用户选择或者其他方式，实施妨碍、破坏其他经营者合法提供的网络产品或者服务正常运行的行为。<br>**第二条**：违反诚信原则和商业道德。 |
| **民事侵权**  | 造成服务器瘫痪、带宽耗尽等财产损失。         | **《民法典》第一千一百六十五条**（过错责任原则）：行为人因过错侵害他人民事权益造成损害的，应当承担侵权责任。                                                      |

### 二、刑事责任
刑事责任是爬虫开发与使用者必须严防的红线。以下分为“获取行为”与“使用行为”两个阶段进行法条充实。
#### 1. 爬取行为（非法访问或获取数据）
这一阶段的核心在于“技术入侵”。

| 罪名                 | 核心行为及构成要件                                                       | 法律出处（刑法条文）                                                                                        |
| :----------------- | :-------------------------------------------------------------- | :------------------------------------------------------------------------------------------------ |
| **非法侵入计算机信息系统罪**   | 侵入**国家事务、国防建设、尖端科学技术领域**的计算机信息系统。（注意：此罪不要求造成破坏，**只要“进去了”就构成**）。 | **《刑法》第二百八十五条第一款**：违反国家规定，侵入国家事务、国防建设、尖端科学技术领域的计算机信息系统的，处三年以下有期徒刑或者拘役。                            |
| **非法获取计算机信息系统数据罪** | 侵入上述三类以外的普通系统（如电商、社交平台），**获取数据**或者**进行非法控制**，情节严重的。             | **《刑法》第二百八十五条第二款**：违反国家规定，侵入前款规定以外的计算机信息系统或者采用其他技术手段，获取该计算机信息系统中存储、处理或者传输的数据，或者对该计算机信息系统实施非法控制... |
| **破坏计算机信息系统罪**     | 造成系统崩溃、服务中断（如高并发DDoS效果），或者**增删改**数据。                            | **《刑法》第二百八十六条**：违反国家规定，对计算机信息系统功能进行删除、修改、增加、干扰，造成计算机信息系统不能正常运行...后果严重的。                           |
| **侵犯公民个人信息罪**      | 爬取的数据包含**姓名、身份证、电话、行踪轨迹**等个人隐私。                                 | **《刑法》第二百五十三条之一**：违反国家规定，向他人出售或者提供公民个人信息...窃取或者以其他方法非法获取公民个人信息的，依照第一款的规定处罚。                       |

> **关键司法解释补充**：根据《关于办理危害计算机信息系统安全刑事案件应用法律若干问题的解释》，非法获取支付结算、证券交易、期货交易等金融数据**10组以上**，或身份认证信息**500组以上**，即可认定为“情节严重”（入刑标准）。
#### 2. 数据使用行为（对获取数据加以利用）
这一阶段的核心在于**“数据内容及其用途”**。即便爬取手段本身可能未被追究，但如果利用爬下来的数据进行非法牟利或破坏，将触犯以下罪名：

| 罪名 | 行为描述与适用情景 | 法律出处（刑法条文） |
| :--- | :--- | :--- |
| **传播淫秽物品牟利罪 / 传播淫秽物品罪** | **情景**：爬取色情网站的视频、图片、小说，并搭建新网站展示或打包出售。<br>**区别**：前者以牟利为目的（量刑重），后者不以牟利为目的（但传播范围广也构成犯罪）。 | **《刑法》第三百六十三条**（以牟利为目的）：以牟利为目的，制作、复制、出版、贩卖、传播淫秽物品的...<br>**《刑法》第三百六十四条**（不以牟利为目的）：传播淫秽的书刊、影片、音像、图片或者其他淫秽物品，情节严重的... |
| **侵犯商业秘密罪** | **情景**：爬取竞争对手后台非公开的**客户名单、定价策略算法、独家货源信息**等，并用于自己经营或公开披露，给权利人造成重大损失。<br>**核心**：数据必须具有“秘密性”（非公开）。 | **《刑法》第二百一十九条**：有下列侵犯商业秘密行为之一，情节严重的...（一）以盗窃、贿赂、欺诈、胁迫、电子侵入或者其他不正当手段获取权利人的商业秘密的... |
| **侵犯著作权罪** | **情景**：爬取小说网站的**付费章节**、视频网站的**独家剧集**、教育平台的**付费课程**，然后建立“盗版站”或低价转卖。<br>**核心**：必须是“以营利为目的”且“未经著作权人许可”。 | **《刑法》第二百一十七条**：以营利为目的，有下列侵犯著作权或者与著作权有关的权利的情形之一，违法所得数额较大或者有其他严重情节的...（一）未经著作权人许可，复制发行、通过信息网络向公众传播其文字作品、音乐、美术、视听作品... |
| **帮助信息网络犯罪活动罪** | **情景**：明知他人利用信息网络实施犯罪（如诈骗、赌博），仍为其提供**数据支持**（如爬取特定人群的联系方式卖给诈骗团伙）。 | **《刑法》第二百八十七条之二**：明知他人利用信息网络实施犯罪，为其犯罪提供互联网接入、服务器托管、网络存储、通讯传输等技术支持，或者提供广告推广、支付结算等帮助... |
### 总结
爬虫技术的法律风险遵循**“手段+内容+后果”**的判定逻辑：
1.  **手段**若涉及突破加密、身份伪造，主要触犯第285、286条（黑客类犯罪）。
2.  **内容**若涉及隐私、国家秘密，主要触犯第253条之一或相关保密法。
3.  **后续使用**若涉及盗版、色情、商业机密，则触犯第217、219、363条等下游犯罪。
